Funcion Suma


La suma o adición de valores en Excel, es una parte integral del análisis de datos, ya sea que esté realizando un subtotal de ventas en la región Noroeste o haciendo un total acumulado de recibos semanales. Excel proporciona múltiples fórmulas que puede usar para sumar datos.

Funcion Promedio


La función PROMEDIO es una función estadística. Se usa para analizar el rango de datos y encontrar el valor promedio.
Volver

Función MIN


La función MIN nos devuelve el valor mínimo de una lista de valores omitiendo los valores lógicos y el texto.
Volver

Función Contar


Use la función CONTAR para obtener la cantidad de entradas en un campo de número de un rango o matriz de números.
Volver

Función Contar.Si


Use CONTAR.SI, una de las funciones estadísticas, para contar el número de celdas que cumplen un criterio; por ejemplo, para contar el número de veces que una ciudad determinada aparece en una lista de clientes.
Volver

Función Contar.Si.Conjunto


La función CONTAR.SI.CONJUNTO aplica criterios a las celdas de varios rangos y cuenta el número de veces que se cumplen todos los criterios.
Volver

Funcion BuscarV


Use BUSCARV cuando necesite buscar elementos en una tabla o en un rango por fila.
Volver

Funcion Coincidir


La función COINCIDIR busca un elemento determinado en un intervalo de celdas y después devuelve la posición relativa de dicho elemento en el rango.
Volver

Función Indice


La función INDICE devuelve un valor o la referencia a un valor desde una tabla o rango.
